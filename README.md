## python -V  
Python 3.9.10（Python 3.6.8がM1 Macの最新のOSで対応していなかったためバージョン変えました。）  

## venvで仮想環境を作成  
$ python3 -m venv .venv  

## 仮想環境に入る  
$ source .venv/bin/activate  

## テストファイル実行  
$ python practice.py
